### Added

## [1.1.0] - 2019-04-10

### Added
- Added go.mod file. [#10](https://github.com/elastic/go-seccomp-bpf/pull/10)
- Added new syscalls to be in sync with Linux v5.0. [#11](https://github.com/elastic/go-seccomp-bpf/pull/11)

### Fixed
